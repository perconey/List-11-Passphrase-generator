# List-11-Passphrase-generator
Passphrase generator allows you to generate custom passwords with settings defined by you.  
For now it has following features:  
-Numeric psw  
-Word psw  
-Alphanumeric psw  

**Custom settings**  
-Lenght  
-Syntax  
-Prefixes, postfixes  

**Additional features**  
-Provide database and generate psw
